# Outside-In Due Diligence Tool — Frontend

React application for workforce and financial analysis, built as part of the AI-Assisted Delivery Experiment (Team 2).

## Overview

This tool enables practitioners to upload workforce data from LinkedIn exports and retrieve financial data from public 10-K filings, combining both datasets into integrated visualizations for Outside-In Due Diligence engagements.

## Features

- **Roster Analysis** — Upload an Excel file and explore headcount by function, seniority, and country, plus full labor spend breakdown
- **Financial Data Pull** — Retrieve Revenue, EBITDA, and Expenses from the latest 10-K filing for any public company
- **Integrated Analysis** — Combined views including seniority pyramid, headcount distribution, and revenue vs. labor spend
