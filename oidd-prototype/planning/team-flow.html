API contract — quick reference
EndpointInputReturns
GET /api/health — {"status": "ok"}
POST /api/roster/analyze Excel file (.xlsx/.xls) total_headcount, total_labor_spend, avg_cost_per_employee, by_function, by_seniority, by_country, cost_by_*
GET /api/financials/{ticker} Ticker string (e.g. AAPL) company, fiscal_year, total_revenue, ebitda, operating_income, depreciation_amortization, expense_breakdown

BE runs on http://localhost:8000 · Interactive docs at http://localhost:8000/docs · CORS allows localhost:5173
